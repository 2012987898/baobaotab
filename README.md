## 简介
“宝宝的标签页”是一款简洁而高效的浏览器标签页扩展，旨在为用户提供流畅、舒适的体验。插件通过精心设计的动画和界面，创造出轻松的使用环境。功能精简，专注于提升用户的浏览体验。

## 安装方法
1. 打开浏览器并进入扩展管理页面。
2. 启用“开发者模式”。
3. 点击“加载已解压的扩展程序”。
4. 选择解压后的插件文件夹进行加载。
